1. 逻辑与结构类（最符合你提到的 Schematic）
这类词侧重于表达事物之间的关系、逻辑或流程。
Diagram: 最通用的词。指为了解释某种原理、结构或流程而画的图（如：Class Diagram 类图）。
Schematic: 原理图/示意图。侧重于逻辑连接，忽略物理细节。
Flowchart: 流程图。专门用于表示步骤、决策路径（在算法逻辑中常用）。
Graph: （数学/数据结构）图。由节点和边组成的结构。在 Java 算法中，Graph 特指处理这种复杂连接的数据结构。

2. 数据与统计类
这类词侧重于数值的变化、比例和比较。
Chart: 图表。用于展示数据的综合性词汇（如：Pie chart 饼图）。
Plot: 散点图/绘图。常用于科学计算，将坐标点画在纸上。
Histogram: 直方图/柱状图。

3. 图像与视觉内容类
这类词侧重于视觉感知。
Image: 图像。泛指任何数字化或看到的画面。
Picture: 照片或画作。通常指写实的、具体的画。
Illustration: 插图。通常是为了装饰或进一步解释文字内容的绘画。
Figure: （书籍/论文中的）插图/编号图。例如你在教材里看到的“Figure 1.1”。

4. 建筑与工程类
这类词侧重于制造和施工。
Blueprint: 蓝图。详尽的、可直接用于施工的尺寸图。
Layout: 布局图。描述物体在空间位置上的排列方式。
Sketch: 草图/素描。快速勾勒出的简略图形，不要求精确。
